# organ-donation
mini project on organ donation and matching system

#Description
The organ donation and matching system is a c language mini project 
that helps hospitals and donation centers store,manage,and match organ
donors with recipients efficiently.It reduces manual work and improves
the speed and accuracy of organ allocation.
#Features
Stores donor and recipient details using structures
Matches based on **blood group**,**organ type**,and **location**
Handles upto 200 records using file handling
Includes **organ priority**,**urgency level** and **availability status**.
Uses c concepts like**arrays,loops,strings,pointers,functions,unions,structures** and **files**.

#Matching logic
A match occurs when:
-Blood group matches
-organ type matches
-optionally city matches

#Future enhancements
